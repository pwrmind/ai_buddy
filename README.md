# 🎙️ AI Buddy: Локальный Hands-Free Голосовой Ассистент

Полностью локальный, приватный голосовой ассистент, работающий без интернета на вашем ПК. Программа автоматически слушает вас (определяя тишину), мгновенно распознает речь на видеокарте, передает запрос в большую языковую модель и озвучивает ответ голосом.

## ✨ Особенности

*   **Hands-Free режим (Рация)** — вам не нужно нажимать кнопки. Скрипт сам понимает, когда вы начали говорить и когда замолчали.
*   **Аппаратное ускорение CUDA** — распознавание речи происходит за миллисекунды благодаря движку `faster-whisper` на базе вашей видеокарты NVIDIA.
*   **Память диалога** — ассистент помнит контекст беседы и может поддерживать полноценный, последовательный разговор.
*   **Умные стоп-слова** — защита от ложных срабатываний. Команда на выход сработает только при явном прощании (фразы вроде *"пока я хочу обсудить..."* не прервут диалог).
*   **Полная приватность** — все данные (звук, текст, история) обрабатываются исключительно на вашем локальном компьютере.

## 🛠️ Стек технологий

*   **Интеллект (LLM):** [Ollama](https://ollama.com) (в связке с моделью `gemma4:e4b-it-q4_K_M`)
*   **Распознавание речи (STT):** `faster-whisper` (модель `base` на движке `ctranslate2` с поддержкой CUDA)
*   **Синтез речи (TTS):** `pyttsx3` (интеграция со встроенным движком Windows SAPI5)
*   **Менеджер пакетов:** `uv` (от Astro) для быстрой и изолированной работы с окружением.

## 🚀 Быстрый старт

### 1. Требования
*   Установленная и запущенная [Ollama](https://ollama.com).
*   Скачанная модель в Ollama (по умолчанию используется `gemma4:e4b-it-q4_K_M`).
*   Видеокарта NVIDIA с поддержкой CUDA.
*   Установленный менеджер пакетов `uv` (или стандартный `pip`).

### 2. Клонирование и установка зависимостей
Склонируйте репозиторий и установите все необходимые библиотеки одной командой:

```bash
# Установка библиотек через uv
uv pip install ollama faster-whisper pyttsx3 pyaudio nvidia-cublas-cu12 nvidia-cudnn-cu12
```

> **Примечание по PyAudio:** Если у вас возникла ошибка при установке `pyaudio` на Windows, используйте `pipwin` или скачайте скомпилированный `.whl` файл для вашей версии Python.

### 3. Запуск
Убедитесь, что приложение Ollama активно, и запустите ассистента:

```bash
uv run main.py
```

## 🧠 Особенности работы под Windows (Решение проблем)

Проект содержит встроенные исправления для специфичных багов Windows:

1.  **Поиск DLL-файлов CUDA:** Внутри скрипта реализован автоматический поиск библиотек `cublas64_12.dll` и `cudnn_ops_infer64_12.dll` внутри изолированного виртуального окружения `.venv`. Скрипт сам регистрирует их в `PATH` процесса до инициализации нейросети.
2.  **Зависание звука SAPI5:** В стандартном `pyttsx3` циклическая озвучка часто ломается на второй реплике. В данном коде движок переинициализируется и уничтожается (`del engine`) при каждом вызове `speak()`, что гарантирует бесконечный и стабильный поток речи.

## 🎛️ Настройка под себя

Внутри файла `main.py` вы можете изменить ключевые параметры в блоке `# --- НАСТРОЙКИ ---`:
*   `MODEL_NAME` — имя вашей модели в Ollama.
*   `SILENCE_THRESHOLD` — чувствительность микрофона (по умолчанию `500`). Если ассистент реагирует на эхо из колонок или шум кулеров, поднимите значение до `800`–`1000`.
*   `SILENCE_LIMIT` — сколько секунд тишины ждет программа (по умолчанию `1.3`), прежде чем отправить вашу фразу на обработку.
*   `engine.setProperty('rate', 180)` — скорость чтения голоса ИИ.
